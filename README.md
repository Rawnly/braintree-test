# Braintree Test
> Braintree nodejs setup

## Installation
```sh
    # Clone this repo
    git clone https://github.com/Rawnly/braintree-test
    
    # Enter in the project folder
    cd braintree-test

    # Fill the .env file with your credentials
    cp example.env .env

    # Install dependecies
    npm install

    # Run the server
    npm start
```
